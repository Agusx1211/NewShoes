# Project New Shoes brand assets

The launcher uses one square, text-free mark so it remains recognizable from a
large setup window down to a small taskbar or installed-app icon.

- `project-new-shoes-mark.png`: transparent full-resolution source artwork.
- `project-new-shoes-mark.webp`: transparent web launcher artwork.
- `project-new-shoes.ico`: multi-resolution Windows icon.
- `project-new-shoes-icon-192.png`: transparent web app icon.
- `project-new-shoes-icon-512.png`: transparent high-resolution web app icon.
- `project-new-shoes-apple-touch.png`: iOS and iPadOS home-screen icon.

`../launcher-logo.webp` is the 512 px runtime copy used throughout the browser
desktop. Keep it synchronized with the mark above when updating the identity.
