Steam installation

  1. Install both Command & Conquer: Generals and Command & Conquer: Generals: Zero Hour from The Ultimate Collection. In Steam, set the game language to English before installing or verifying the files.
  2. Open Steam Library, right-click Zero Hour, then choose Manage > Browse local files. If Steam opens a collection folder, continue into the Zero Hour game folder.
  3. Return here, choose Add Game Folder, and select the root that contains the Zero Hour executable, a Data folder, and files such as EnglishZH.big, INIZH.big, and W3DZH.big. The scanner also needs the Generals base archives supplied by the same installation.

EA app installation

  1. Install The Ultimate Collection in the EA app with English selected. Open the game’s Manage or View properties screen to find its install location.
  2. Open that location in File Explorer, find the Zero Hour root, then choose Add Game Folder here. Select the folder containing the executable, Data, and the Zero Hour .big archives, not the parent that only lists every game in the collection.

What happens next: the browser asks for read permission and inventories file names and archive structure locally. Temporary mode asks again next visit. Remember mode retains a browser permission when the browser supports it. Install mode copies only the required archives into this site’s private browser storage. Nothing is uploaded.

If validation fails: verify both Generals and Zero Hour in Steam or the EA app, confirm English is installed, and select the actual Zero Hour root. Modified or repacked releases may rename, recompress, or omit required retail archives and are not supported.

I own the original discs
  1. Create or locate readable images of media you own. This launcher does not bypass copy protection and does not provide instructions for defeating DRM.
  2. Choose Add Disc Images and select the complete English set together: Generals Disc 1, Generals Disc 2, Zero Hour Disc 1, and Zero Hour Disc 2. For BIN/CUE media, keep each matching pair together and select all relevant files.
  3. Review the selected-media list. Add another part if anything is missing, or remove a mistaken file with its Remove button. The local scanner identifies the required archives across all selected parts and reports exactly what is still missing.
  4. After validation, choose Just launch this time, Remember the files, or Install in this browser. Remember keeps permission when supported; Install copies the required archive set to private browser storage and can use several gigabytes.

Only the English retail archive set is supported today. A damaged image, the wrong language, a missing second disc, or an unsupported repack will fail validation. Re-read or replace the affected media from your own copy; do not download individual game archives from third-party sites.
